Markdown 
 در سال 2004 با این ایده که بتواند خواندن ونوشتن یک متن تحت وب را برای همه ساده کند و markdown
قابل تبدیل به خانواده اچتمل هم باشد به وجود آمد.
وبه عنوان یک استاندارد متن نویسی یاد می‌شود و بزرگترین مزیت این استاندارد استفاده از حروف ساده و کوتاه برای اهداف نگارشی (از جمله پررنگ کردن متن و …. ) می‌باشد.
به عنوان مثال برای پررنگ کردن متن با استفاده از مارکدون، کافی است کلمه را بین دو ستاره (*) قرار دهیم.
از دو بخش تشکیل شده است: بخش اول ویرایشگر متنی که نوشته را گرفته و استانداردسازی می‌کند و بخش دوم تبدیل‌کننده‌ای به زبان پرل است markdown
 که نتیجه را به اچتمل تبدیل میکند.
سایت هایی مانند 
GitHub ، Bitbucket ، Reddit ، Diaspora ، Stack Exchange ، OpenStreetMap و SourceForge
از انواع مارکدون برای نگارش آسان و سریع کاربران استفاده میکنند
